זהו פוסט לדוגמה שנכתב ב-**Markdown**. הוא מדגים את היכולת החדשה לכתוב פוסטים בפורמט Markdown במקום HTML.

## יכולות Markdown

* רשימות מסומנות
* **טקסט מודגש**
* *טקסט נטוי*
* [קישורים](https://example.com)

## קוד לדוגמה

```javascript
function hello() {
  console.log('שלום עולם!');
}
```

## ציטוט

> זוהי דוגמה לציטוט בעברית. ציטוטים מוצגים עם רווח בולט.

## סיכום

פורמט Markdown מאפשר כתיבה נוחה וקלה יותר מ-HTML, תוך שמירה על כל היכולות הנדרשות לפוסטים באיכות גבוהה.
